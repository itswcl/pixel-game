## ðŸ› ï¸ Backend Setup (Google Sheets & Apps Script)

This project uses Google Sheets as the backend. Please follow these steps to set it up:

### Step 1: Create a Google Sheet

1. Go to [Google Sheets](https://sheets.google.com) and create a new spreadsheet.
2. Name the spreadsheet `Pixel Quiz Database` (or any custom name).
3. **Create "Questions" Sheet**:
   - Rename the bottom tab/sheet to `Questions`.
   - Set the following headers in the first row:
     `ID` | `Question` | `A` | `B` | `C` | `D` | `Answer`
4. **Create "Responses" Sheet**:
   - Add a new sheet and name it `Responses`.
   - Set the following headers in the first row:
     `ID` | `PlayCount` | `TotalScore` | `MaxScore` | `FirstPassScore` | `AttemptsToPass` | `LastPlayed`

### Step 2: Deploy Google Apps Script

1. In the spreadsheet, click **Extensions** > **Apps Script** in the top menu.
2. Copy the entire content of the `google_apps_script.js` file from the project root.
3. Paste it into the Apps Script editor, overwriting the default code (`function myFunction() {...}`).
4. Click **Deploy** > **New deployment** at the top.
5. Click the gear icon âš™ï¸ > Select **Web app**.
6. Configure as follows:
   - **Description**: Pixel Game Backend
   - **Execute as**: **Me**
   - **Who has access**: **Anyone** âš ï¸ _This is crucial; otherwise, the game cannot read the data._
7. Click **Deploy**.
8. Copy the generated **Web App URL**.

## ðŸ’» Frontend Setup & Execution

### 1. Installation & Setup

```bash
# Install dependencies
npm install

# Copy environment variable example file
cp .env.example .env
```

### 2. Configure Environment Variables

Open the `.env` file and fill in the Web App URL obtained in the previous step:

```env
VITE_GOOGLE_SCRIPT_URL=https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec
VITE_PASS_THRESHOLD=3  # Passing threshold (number of correct answers required)
VITE_QUESTION_COUNT=5  # Number of questions per game session
```

### 3. Local Development

```bash
npm run dev
```

Open your browser and visit the displayed Local URL (usually http://localhost:5173).

## ðŸš€ Deployment (GitHub Pages)

This project is configured with GitHub Actions for automated deployment to GitHub Pages.

1. Push the code to GitHub.
2. Go to the GitHub Repo **Settings** > **Secrets and variables** > **Actions**.
3. Add a Secret: `VITE_GOOGLE_SCRIPT_URL`, and fill in your GAS Web App URL.
4. Go to **Settings** > **Pages**, and set Source to **GitHub Actions**.
5. Any push to the `main` branch will simply trigger a deploy.
